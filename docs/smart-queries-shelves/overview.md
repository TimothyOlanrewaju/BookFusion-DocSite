Uploading books to your BookFusion library is quick and easy. You can upload various file types including `.epub`, `.pdf`, and `.mobi`.

## Supported Formats

- EPUB (.epub)
- PDF (.pdf)
- MOBI (.mobi)
- CBZ/CBR (comics)
